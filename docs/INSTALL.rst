cgso.homepage Installation
--------------------------

To install cgso.homepage using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``cgso.homepage`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        cgso.homepage

* Re-run buildout, e.g. with:

    $ ./bin/buildout
